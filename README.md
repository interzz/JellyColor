# Heroku Модуль для Создания Премиум Эмодзи

Этот модуль предназначен для создания премиум эмодзи с пользовательским текстом. Он предоставляет функционал для создания и настройки уникальных эмодзи, а также их перекраски.

## Описание

Модуль, который позволяет создавать эмодзи с индивидуальным текстом, а также изменять их цвета. Несмотря на некоторые мелкие баги, он работает достаточно стабильно, учитывая, что был разработан за короткий срок.

### Основные особенности:
- **Создание эмодзи** с вашим собственным текстом.
- **Перекраска эмодзи** для индивидуализации.
- Легкий и быстрый в использовании.

### Примечания:
- Модуль был создан за полдня.
- Присутствуют некоторые мелкие баги, что естественно для такого быстрого прототипирования.
- Это был первый опыт с lottie и tgs
