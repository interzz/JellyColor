# Heroku Модуль для Создания Премиум Эмодзи который стоил 100 бекесов

модуль для создания премиум эмодзи с пользовательским текстом. 

## Описание

модуль который позволяет создавать эмодзи с кастом текстом

- Модуль был создан за полдня.
- Присутствуют некоторые мелкие баги, что естественно для такого быстрого прототипирования.
- Это был первый опыт с lottie и tgs
